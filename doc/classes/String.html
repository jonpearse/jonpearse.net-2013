Returns a normalised version of the string, where all UTF-8/accented/weird characters are converted to their nearest ASCII equivalents. This requires the Babosa gem to be installed and available.

def slugify
  normalize.gsub(/[^\sa-z0-9\-\_]/,'').gsub(/\s+/,'-').gsub(/--+/, '-')
end
